## ✨ Features

- **Zero dependencies** — pure HTML, CSS, and vanilla JS. Open `index.html` and it works.
- **Fully responsive** — mobile-first layout that looks great on all screen sizes
- **Smooth scroll animations** — intersection-observer based reveal on scroll
- **Project filter** — live filtering by category (APIs, Data, Auth)
- **Working contact form** — powered by Formspree, no backend required
- **GitHub contribution graph** — visual activity heatmap
- **Dark professional theme** — deep navy palette with accent highlights
- **Performance optimised** — no frameworks, loads in under 1 second
- **SEO ready** — meta tags, Open Graph, and Twitter Card support

## 🚀 Running Locally

No installation required. Just serve the files over HTTP:

```bash
# Option 1 — Python (built-in, recommended)
python3 -m http.server 8000

# Option 2 — Node.js
npx serve .

# Option 3 — VS Code
# Install the "Live Server" extension → right-click index.html → Open with Live Server
```

Then visit **http://localhost:8000** in your browser.

> ⚠️ Always use a local server rather than opening `index.html` directly as a file (`file://`). Some browsers block font loading and form submissions on `file://` URLs.
